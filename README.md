# UIImageView
简单好用的仿ios的点击变暗效果的按钮！！！
一般Android中实现一个ImageView的点击效果需要两张图片，每次美工切图麻烦不说，
也会增加app的体积,该控件只需一张图片就可以实现点击变暗效果。
